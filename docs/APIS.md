# APIs description

### Health Check

Request

```
GET /date-time/v1/health
```

Response

```
{
  "status": "UP",
  "date-time": "2016-11-09 19:40:33",
  "hostname": "server-machine-test",
  "version": "1.0-SNAPSHOT"
}
```
